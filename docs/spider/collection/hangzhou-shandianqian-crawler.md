### 4. Python 与网络基础
**爬虫负责人：** Cookie 和 Session 有什么区别？爬虫中怎么使用 Session？

**候选人：** Cookie 通常保存在客户端，可以持久化；Session 通常保存在服务端，并可能按后端设置过期。爬虫里会用 Session 在一段连续请求中保持登录状态、权限和部分请求上下文。

**爬虫负责人：** HTTP 和 HTTPS 有什么区别？

**候选人：** HTTP 默认是明文传输；HTTPS 在 HTTP 上增加 TLS，通过证书进行身份验证，并对传输内容加密。

**爬虫负责人：** 浅拷贝和深拷贝有什么区别？

**候选人：** 浅拷贝创建新的外层对象，但内部嵌套对象仍可能与原对象共享引用；深拷贝会递归复制内部对象，修改副本通常不会影响原对象。对于整数、字符串、元组等不可变类型，表现会有所不同。

**爬虫负责人：** 多线程和多进程有什么区别？

**候选人：** 进程是资源分配的基本单位，线程是 CPU 调度的基本单位。一个进程可以包含多个线程。Python 中，多线程更适合 I/O 密集任务；CPU 密集任务通常更适合多进程。原始回答中我把两者关系说反了，这里予以更正。
