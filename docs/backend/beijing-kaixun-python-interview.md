## Python 基础
**面试官：** Python 做了大概多长时间？

**我：** 从大二开始就用 Python 写一些小脚本。2024 年 6 月本科毕业后到现在，也一直在使用 Python 开发。Scrapy、正则匹配、FastAPI 这些都是工作中用到的。

**面试官：** 好的，我先问一些 Python 基础。Python 有一些内置函数，`map` 和 `filter` 用过吗？

**我：** `filter` 是用来过滤迭代器中元素的。`map`……是 `dict` 吗？

**面试官：** `map` 和 `filter` 都可以处理数据。`map` 就像映射，例如有一组数字，想让每个数字都乘以二。

**我：** 哦，这个内置函数确实很少使用，有点忘了。

**面试官：** Python 的基础类型中，哪些可变，哪些不可变？

**我：** 首先元组是不可变的。另外可能还有一些常量，我忘记 Python 有没有常量类型了。其他的应该都是可变的。字符串可变吗？字符串好像是不可变的，但可以进行各种切片；每次操作实际上都会生成一个新的字符串。整数应该也是不可变的，每次变化相当于产生一个新值。

**面试官：** `int` 也是不可变的。Python 是面向对象的，魔术方法 `__new__` 和 `__init__` 哪个先执行，哪个后执行？

**我：** `__new__` 应该是类层面的，用于创建实例；`__init__` 是实例层面的。因此 `__new__` 在前，`__init__` 在后。

**面试官：** 函数传参时，`*args` 和 `**kwargs` 都可以传多个参数，它们有什么区别？

**我：** 一个好像传列表，另一个传多个参数。可能一个参数中有多个值，另一个是多个参数、多个值。

**面试官：** 一个接收元组，一个接收字典。

**我：** 哦，一个是元组，一个是字典，相当于进行了解包。

**面试官：** 装饰器中的 `staticmethod` 和 `classmethod` 有什么区别？

**我：** `staticmethod` 是静态方法，属于类层面，不实例化对象也可以使用。`classmethod` 可能需要实例化，通过对象使用。

**面试官：** 你刚好说反了。`classmethod` 不需要实例化对象，可以直接通过类调用；`staticmethod` 也是静态方法，在类中可以直接调用，不需要创建对象。

**我：** 明白了。

## Django、ORM 与 Redis
**面试官：** Django 用过吗？

**我：** 用过。

**面试官：** Django 的设计模式是什么？

**我：** 好像叫 DRF……不对，DRF 是它的一个框架。好像是 MVT。

**面试官：** 对，MVT。

**我：** Model、View 和 Template。

**面试官：** Django 的 ORM 用过吗？

**我：** 用过。它有自己的一套 ORM。

**面试官：** 它的 ORM 是惰性的还是非惰性的？

**我：** 是惰性的，QuerySet 是惰性执行的。

**面试官：** Redis 用过吗？

**我：** 用过。Redis 是内存数据库，读写特别快。它也是非关系型数据库，可以存储键值对。

**面试官：** 它有哪些数据类型？

**我：** 有序列表，然后可能有数字……没有数字。我经常用的是列表。我之前开发过一个代理池，数据就是存在 Redis 里的。

**面试官：** Redis 有字符串、哈希、列表、集合和有序集合这五种常见类型。Redis 持久化做过吗？

**我：** Redis 持久化应该可以通过配置实现。

**面试官：** 它有两种持久化方式，接触过吗？

**我：** 目前没有接触过，应该可以通过配置实现。

**面试官：** 一种是 RDB，相当于做快照；另一种是 AOF，把执行的命令追加记录下来。

**我：** 明白了。
