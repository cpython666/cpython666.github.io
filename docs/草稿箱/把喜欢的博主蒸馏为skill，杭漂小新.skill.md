把喜欢的博主蒸馏为skill，杭漂小新.skill

很早以前我就在尝试使用OCR或者是语音识别的办法去获取视频文案，当时我就在想把一个博主的所有文案处理成文本，然后让ai学习，类似知识库的样子 
然后就可以复刻一个博主的理念，语气  
就相当于有一个私人教师。  
现在我觉得已经可以初步实现这样一件事情了。就是把博主的风格浓缩为skill。   
因为我的视频大部分都是有文案的，所以我就让ai给我的文章风格浓缩为skill，这样我想写某个话题的文章之前就可以把我的草稿，一些零散的想法发给CodeX，让他用我的写文章skill给我写文章。  
我试着让他写了两篇文章，感觉写的还挺好的。很多观点和我想的一样。
