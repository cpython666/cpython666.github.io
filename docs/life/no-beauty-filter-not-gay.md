# 不开美颜，是一种筛选，我不是gay也是

哈喽观众们好啊，这期视频的主题是啊：不开美颜，也是一种筛选

就是最近啊，有个观众建议我：“哥，你其实可以开个美颜。”什么意思？这意思不就是说我长得不行吗？我一怒之下怒了一下，反问他：“你以为美颜救的了我吗？”

这个我其实之前也思考过，要不要开美颜，最后一番权衡之后就没有开。首先呢我先来对比下我开不开美颜的区别，用的是抖音的摄像，可以明显感觉开了美颜看起来是更好点的。《直接从一个修空调的变成坐办公室的》，《但是我比较好奇，开了美颜线下还能有人认出我吗。》

下面就讲讲我对这个要不要开美颜的思考。

首先开美颜有哪些优点呢？就有人认为啊：就是我们其实在一个很现实的世界啊，如果你外貌出众，那么你的机会无论是就业还是各方各面和人打交道的事情上都会有点优势。在生活中，你的外貌吸引来的别人的关注不仅会提高一些事情上的优势，并且会潜移默化的影响你的信心。在潜意识和现实的各种奇妙的反应后，一个人获得成功的概率可能就更大了。有句话也说：现在大家都是外貌协会的，没五官，谁了解你的三观。外貌好可以吸引读者阅读兴趣，确实啊，正所谓兵马未动，广告先行。认不认真读先给吸引过来。毕竟我刷短视频三秒之内看不到美女就下一个了。

之前好像也是有那种社会学实验啊，就是让老师对于一些学生表示肯定和夸奖，投入额外的注意。发现这些被特殊正向对待的孩子成绩可能就更好，就更礼貌。大致是这个意思。

但是呢，我知道了上面这些道理之后啊，依旧觉得还是不开美颜比较好。

首先因为我对我的定位就是技术，生活，人生哲理分享型博主。工作也不太需要颜值。平时写文案感觉已经消耗我足够多的精力了，顾不上再给视频加美颜，注意各种细节。

其次啊，我觉得不开美颜也是一种筛选，筛选掉了关注外貌的人。我不可能开了美颜，哪期视频没开美颜了掉粉了。或者说，这里打个比方啊，有个女网友找我，聊半年，最后见面发现：“你怎么和视频里长得不一样？”，然后爱情结束得像互联网公司倒闭一样突然。

有个故事啊，我记不太清楚了，就是有个诈骗手段啊，就是给人群发信息说：“我是秦始皇转世，v我转500启动复国计划。”其实正常人一看都会觉得很搞笑啊，这种东西谁会信啊。但结果呢，被骗的人还不少。为什么呢，因为这种简单的骗局本质也是个筛选，如果你连这种低级话术都信的话，就说明你是个笨蛋。你被骗的概率就很大。骗子直接把那些不好骗的聪明的给筛选掉了，直接针对好骗的交流，一定程度上还节约了人力。听了这个故事大家有没有觉得很有趣啊，就是筛选出你的目标人群。这就是第一点，筛选出我的目标人群。

那么有兄弟就说了，啊，我懂了。原来这是筛选目标人群啊然后跑到人多的地方大喊：我是gay！

谁笑谁是gay。

那又有兄弟说了：我不是gay啊。你不是gay也是gay，因为此地无银三百两。

言归正传。第二点就是用外貌获取自信心的好处，我承认确实有。但我后来发现，别人夸你帅五分钟，房租还是要交。别人说你有气质，老板也不会因此给你涨工资。刚才还是开玩笑。主要我觉得我个人其实不太需要别人的肯定，我知道自己需要的是什么，可以追求自己内心的宁静。没有别人的认可我也可以，有的话我也会很开心。我不需要你对我的外貌多么肯定，我自己就已经把自己洗脑成绝世帅哥。好看可以吸引别人的注意，我长得好笑一样可以。所以呢我也不需要美颜。

最后声明啊，我的不开美颜并不代表彻底放弃治疗完全不在乎外貌了，我还是会努力一下的，正常还是早晚会洗把脸的，保持衣服干净整洁。这期视频呢只是谈谈我的看法。我们下期视频再见。
