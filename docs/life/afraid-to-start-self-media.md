# 想做自媒体不敢开始？不敢露脸怕被别人看到？

![image.png](/imgs/life/self-media-start-fear.png "image.png")

做自媒体怕被熟人刷到怎么办？比如亲戚同事领导或者家人看到。

我是什么看法呢，其实我对这个很有感悟，因为我就是从这种心态开始的。

我最开始的视频就是不敢露脸不敢漏声音，使用ai配音的。就是因为害怕认识的人刷到，有容貌焦虑等。但是这样制作视频特别麻烦，需要画面和文案对应起来甚至是学一些复杂的动画让视频有趣起来。很消耗精力和时间。

所以我就努力去克服自己的胆小，也是经常会想到一些鸡汤，在脑子中回荡，但是真正录好视频看到自己的大脸之后还是没有剪辑和发布的勇气。最后也是一步一步，从录声音，二次元动捕人物，到只敢带着墨镜，再慢慢到现在这样制作视频，我其实也是经历了一年多的跨度。现在我敢于直接这样做之后，也有了更多的感悟。

:::info
第一点是：我觉得害怕别人看到是没必要的，因为你首先你可以设置一下不推荐给可能认识的人，虽然说它的作用不大，但是你可以稍微选一下。

第二点：是我觉得其实在现实中，每个人都有自己的生活，忙着工作、玩游戏、外出，或者沉浸在自己的世界里，根本没有那么多时间和精力去关注你在干什么。就是你不要把自己想得太重要了。怕这怕那的其实没有必要。我最开始做的时候也是非常的胆怯害羞胆小什么的，之前没露脸的时候，发出作品也会有同事问这个人是不是你。目前遇到的只有一个就是问了下是你吗，剩下的就没有了。甚至说我在最近把我的视频分享给我家里人，就几乎没什么动静，转发过去其实就有点像发消息给你已读不回的女神，她看了没你也不知道，反正就是没有任何的回应。

第三点：害怕朋友同事看到在我身上几乎没有，因为目前就我自身而言，没有人跟我说，哎呀，我看到你啥你自媒体怎么怎么着。然后你做自媒体发视频呢，觉得你说的好的呢，自然就会经常看你的视频。有的可能觉得哎我生活中见到你的人已经够烦了，怎么我刷视频还能看到你，而且发的是什么东西，一点也看不懂，可能就点个不感兴趣或者是拉黑了。大家也都是匿名上网的，不发作品的你看头像昵称你也认不出人啊。如果是也发作品的话你们就是同道中人那就更不用怕了，因为大家都抛头露面的，可能还会心有灵犀。有时候被领导看到了可能还是加分项，可以看出你对某件事情的热爱。

第四点：然后有的不敢露脸可能是因为容貌焦虑，我就有容貌焦虑，但是后来我也看开了，我就长这样，你能接受接受，接受不了我也没办法吖。我觉得做自媒体就要心大，要学会接受一切。我最开始的时候视频发出去几天了可能都觉得后悔，可能睡一觉起来又把视频给删了，包括现在，我都没有勇气去看我之前有些视频。就当是记录生活了，我相信不同的时间都会有不同的感受。

而且我觉得我的容貌也是充满了缺点，但是我在这里不想展开说了免得会显得我很自卑。而且互联网本来就是包容的，各种各样的都有，不是说只能有颜值和智商才能成功，也有很多土味或者是被喷的赛道。所以说颜值并不是成功的必须因素。再说很多你看到的很精致很帅，很美的照片，视频部分原因是因为化妆，拍摄的设备比较好再加上后期。

第五点：其实本来大家来互联网上可能就是为了搞钱或者是找乐子的，但是你却非要认真，并且还把自己假设在一个非常重要的地位，这本身就不是很合理。所以容貌焦虑其实是不必要的。

第六点感悟是：网上之前有一个说法是什么？在应该摇花手的年龄选择了读书，表示很后悔。其实我觉得不一定，因为当初那些人就是在看不到挣钱可能的情况下就选择了去摇花手，但是很多人呢，就算现在知道摇花手是挣钱的，可能也不一定会去摇手。为什么呢？因为把自己看的太重要，因为害怕被别人看到而不敢开始，就是想很多，但是做得很少。我的建议是不要想那么多，要想的话这个问题是无穷无尽的，所以说你尽管做就把心思放在怎么做好你的作品怎么开始？想的时候全是问题，只有做才会有答案。比如说你做之前会想：被熟人看到了怎么办，但是当你开始做之后，你就不会有这个烦恼了，因为可能根本没人看你的作品，这个时候你的烦恼又变成了哎为什么没人看我的作品。

然后有时候呢你可能会给自己找借口，说什么红利已经过去了，说自己错过时代了什么的，但是其实你如果你根本就没有开始的话，没有具备勇敢等特质的话，那么就那个风口来了，你一下就能抓到他吗？我想概率会比较小。更大可能是你刚好已经开始了，在路上，然后也碰到了风口，那么你才有抓住风口的可能。而不是像等一场入室抢劫般的爱情一样，等着风口来了，吹到你家，把你从床上吹到天上。

所以总结下来我做自媒体是什么心态呢？我就是记录生活，分享交流观点，来学习和交志同道合的朋友的。我长得就这样，夸我帅了我开心，说我丑了我也无所谓。喜欢我视频了平台会自己推送给你，不喜欢了点不感兴趣或者拉黑我选择权在于你，并且不会对我有任何影响。我就是我，我就在这里，无论你喜欢与不喜欢，看与不看，我都在这里。做自媒体，不要让担忧和自我怀疑限制了你。开始，享受创作过程，结果自然会随之而来。

最后再加一句刚才刷到的文案：改变自己最快的方式，就是做自己害怕的事情，勇士脚下没有绝路，懦夫眼前尽是悬崖。

好，这集分享就到这里，感谢大家的观看。如果对我的后续作品感兴趣，可以点个关注。
