# 推荐两个我最近反复看的博主：陈昌文、路飞

这个视频给大家推荐两个我最近一直在看的博主，一个叫做陈昌文，一个叫做自己造船的路飞。

首先这两个人他们特别的有钱，当然我不是说谁有钱我就关注谁、觉得谁说得好，而是他们有钱还真诚，没有架子。他愿意把它领悟的东西给讲出来，并且最重要的是能和我产生共鸣。

他们的视频没有多么炫酷华丽的封面或者词藻，没有专门布置的场景，可能就是怼着脸就开始说了。

他们讲的东西我这里三两句话形容不出来，大家感兴趣的话，随便翻他们两三个视频看一下，喜欢就是喜欢，不喜欢就是不喜欢。

反正我就是他们的切片视频我刷到十个会有八个给看完，我觉得他们说的都是实话，然后觉得说的很对说我心坎上的感觉。

其实现在互联网上的信息非常的多，你可以看到很多那种搬运的，抄袭的或者AI生成的就没有感情的快节奏产品，不知道大家懂不懂那种感觉，第一次看到还好，但是后面再刷到就会很烦。

这个视频就是分享下我很喜欢的这两个博主，希望大家也会喜欢吧。
