# 做自媒体，就是在向世界发射信号。想赚钱，先修心

经常有朋友问我怎么赚钱，我的回答第一推荐永远都是去做自媒体

用吸引力法则来讲就是吸引与你想法一致的群体，散发出你的魅力，让这个世界感受到你的存在，然后跟你相近的人才会被吸引来

大部分人其实没有内容去做自媒体的，要做的话你就的反思自己有什么内容，有什么东西，有什么特长可以拿出来给大家看。做的过程中要克服自己的自卑恐惧在镜头前面展现自己。这个过程其实也是认清自己接纳自己的过程。然后你就变得勇敢和通透起来了，这是很难得的。

那么问题回到了怎么赚钱上面，我断断续续做了三四年自媒体其实没赚一点钱，那三五百块钱其实不如我把做视频的时间用来多睡觉。我没有得到结果我觉得有多方面原因：比如我做的赛道，形式，主题等，我也没有坚持很久。

但是做自媒体潜移默化对你的提升可以增大你赚钱的能力或者说概率。不一定体现在视频收益里，因为你在做自媒体的过程中，你的社交圈子会增大，你遇到的机会会增多，这些都是收益，而且是一个厚积薄发的积累，所以说我目前有开始做自媒体这件事情，换了一种风格，展示更加真实的我，分享一些所思所想。

上面就是我的看法，大家仁者见仁。
