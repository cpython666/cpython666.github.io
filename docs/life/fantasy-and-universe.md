# 那时候的我，靠幻想和宇宙活着

### 优化后的视频文案（更适合口播节奏）
在我高中那段最灰暗的时候，  
有两本书，对我的影响特别大。

第一本，是 三体。

我第一次意识到，  
原来人类、文明、宇宙，  
可以被想象得那么宏大。

它让我在现实压得喘不过气的时候，  
把目光投向了更遥远的地方。

后来我发现，  
有时候人需要的，  
不是立刻解决痛苦，  
而是先从痛苦里抬头。

还有一本书，叫 秘密。

很多人可能没听过这本书，  
但一定听过一个词——  
吸引力法则。

它讲的东西其实很简单：  
你持续相信什么，  
你就会慢慢靠近什么。

当时的我看完特别震撼。

因为它第一次让我觉得：  
“原来一个人，  
可以先相信自己，再慢慢成为自己。”

当然，后来我也意识到，  
这本书有点像“精神鸡汤”，  
甚至有点像“洗脑”。

但现在回头看，  
我依然觉得——

盲目的自卑，  
还不如盲目的自信。

至少自信的人，  
会真的往前走一步。

那几年里，  
一本书让我看见了宇宙，  
另一本书让我重新看见了自己。
