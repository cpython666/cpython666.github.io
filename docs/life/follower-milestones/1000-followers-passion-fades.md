# 【千粉纪念】一千个粉丝了，激情不再！

本来并没有打算做千粉纪念的视频，直到在首页偶然看到了别人的千粉记录。那一刻，仿佛看到了曾经的自己，就是那种做视频收获粉丝的激情，这让我不禁让我意识到，我已经很久没有这样的总结与反思了。

回想起500粉丝时的纪念，当时内心充满了激情，迎接着期待已久的500位支持者。我精心制作了纪念视频，但如今，1000粉丝了，最大的变化是什么呢？正如我一开始所言，甚至都不再那么强烈地想要做一个纪念视频。

我觉得这个过程中最大的变化，正如标题所写：激情不再！

曾几何时，我的粉丝数量还只是个位数，其中甚至包括我的小号。每次发布视频后，我都会切换到小号，反复观看自己的作品，然后点击“三连”。

曾几何时，我的粉丝还只有两位数，每增加一个粉丝，我都会点开他们的主页，以示尊重。我频繁地刷新页面，最开心的时刻就是看到粉丝数量增加，最担心的时刻则是看到数量减少。

曾几何时，我的粉丝数量达到了三位数，每天醒来的第一件事就是打开哔哩哔哩的个人页面，下滑刷新。我开始怀疑自己的努力是否有意义，但也曾经历了几次视频播放过万，带来粉丝暴涨的喜悦。

而现在，我的粉丝已经达到了1000，已经进入了四位数。在这个过程中，我经历了许多支持，也遭遇了不少质疑和批评。

然而，正如我的标题所写：激情不再！

究竟多少粉丝才算是成功？谁的认可才算是我的追求？也许在一次次的得失中，我渐渐地释然，渐渐地领悟。

一千个粉丝，已经超越了做自媒体的许多人。但在人海之中，一千个粉丝又显得微不足道。究竟以谁为标准，这是一个难以回答的问题。

我的答案是什么呢？我还没有想清楚。

但我唯一确定的是，不要停止前行！

## 格言

前进的道路上，激情或许会消退，但坚定的信念将永不磨灭。

为什么不去奔跑？

奋斗的重点是无可奋斗。
