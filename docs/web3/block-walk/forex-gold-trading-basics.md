# P10-做外汇黄金的常识，算了下我交易过几个小目标的黄金

先讲一些基础知识，后面讲一些数据的细节，因为基础知识你不知道踩了坑就是竹篮打水一场空，数据算不对只是操作的多了多花点手续费。

第一点，只用前三的经纪商，前三是啥我也不知道，我用的是第二的叫TMGM。小的会跑路，会卡盘，会出金困难，不给出金。不要用小平台！小平台可能你的交易不真正下单，只是数据盘。因为散户韭菜大部分人炒股没杠杆都是十赌九输，外汇最高一千倍杠杆，能赚的人百不存一。人家只用接个大所价格数据，散户充钱进去随便玩就是亏光光，充钱就是给人家送钱。

第二点，一般会有人给你晒单，全是赚的吃大肉，信的人都是蠢驴。这种人会给你邀请链接，说用它链接注册可以进群，用他的连接注册了你有交易量手续费会有一部分给他是返佣，具体返佣有个例子后面讲。群里点位有亏有赚，就算胜率高，也不值得你浪费那个时间和精力天天看着，不如去学个技能打个工。赚钱还是要赚自己认知以内的，跟着别人赚的钱也不是自己的，迟早靠自己的实力亏回去。还有那种卖课的卖量化软件的本质是因为卖课风险小于他自己去做交易的风险。毕竟市场永远都在，杠杆也可以开很大，真的厉害就可以像提款机一样从市场提钱了谁还教你赚钱，多用脑子想想为什么。本质还是猜大小，就算理论对但是做交易从来不是纸上谈兵，需要耐得住寂寞，管得住手。学会了理论大概率你也知道做不到。

第三点，mt5，mt4只是工具，想做量化电脑端的软件提供有接口可以用python调用获取k线和下单。现在ai写程序很厉害的。但是量化很容易过拟合。比如你回测出了一个收益率胜率很高的参数，但其实你这个参数就是过拟合的。你是不知道未来的市场那个哪个参数更好。

第四点，之所以能盈利是因为仓位管理的好，能管得住手不乱开单，选择赢面大的。不是因为他能预测走势，不能盲目信任，前面赚小的后面一波亏个大的。想要赚快钱的必定遭到反噬。

然后就是手数的知识。国内金价按克计价打个比方是1000月每克，国外是按盎司计价打个比方是5000美元每盎司，然后一手是100盎司，也就是50万美元一手。

我之前用的TMGM，没有手续费，有点差，点差32。点差就是买的价格比实际价格贵，比如说一盎司黄金就是0.01手，点差就是0.32。就是黄金价格5000的时候你买入，买入价格就是5000.32。也就是做一手黄金手续费差不多32刀。然后我当时使用的一个人的邀请链接，那个人设置的返佣客户点差12，也就是说实际做一点差是20，就是我做一手的手续费是20刀。当然你用一些人的邀请链接大概率他是不会给你返佣的，都揣自己的腰包里了，这就是很多人发自己的邀请链接想让你用的原因。

在外汇黄金上我陆续是充值了4000刀，返佣我返了1000刀。当时我的交易量还是比较大的，大致算了下我交易量差不多100手，我去，还是有点震惊。5000万美元，差不多3个多亿人民币的黄金的交易量。点差换算的手续费差不多都两三千刀。亏掉真活该我啊。

因为黄金波动小，所以这么大交易量应该也不稀奇。
