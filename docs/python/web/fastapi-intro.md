## 为什么这个合集？
+ 前面的数据采集中间系统就是用的fastapi，如果后面要支持更多功能，比如说限流，登陆，数据库相关的功能
+ 三四行代码就启动一个服务【特别特别简单】，看文档就可以学习
+ 文档可以学到基础，但是还有很多需要设计好，还有很多工具需要封装
+ 网站正在使用FastAPI重构，应为我对于Django很多东西都是一知半解【drf，权限认证这些】，用起来很不自在，
+ 这个合集就会分享一下涉及这些东西，封装好的代码大家可以直接copy，这就是这个合集的价值
