# P2-提取视频文案的几种场景及其优缺点-字幕提取器

[P2-提取视频文案的几种场景及其优缺点-字幕提取器](https://www.bilibili.com/video/BV1UN411L7fU/)

以下是一些常见的提取视频文案的场景以及它们的优缺点：

1.  **语音识别技术：** 
    - **场景：** 无字幕
    - **缺点：** 对于嘈杂的背景音、口音较重的说话者，识别准确率可能下降。依靠模型好坏，电脑的性能
2.  **图像识别和OCR技术：** 
    - **场景：** **** 适用于字幕准确的视频
    - **缺点：** 识别时长，字幕颜色区分度，
    - **改进**：抽帧，比如每隔一秒，30分钟视频只需要30*60张，最后再只截取视频底部的字母区域，识别应该不是很难。
