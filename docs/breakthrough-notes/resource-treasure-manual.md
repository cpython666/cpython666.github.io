# 找资源宝典：资源无界，打破信息壁垒-说明

会员可见：找资源宝典：资源无界，打破信息壁垒

人与人之间的智力差别起初可能并不大，后期的差别很大可能由于信息差和执行力。

本宝典旨在教会大家如何在资源浩如烟海的互联网中快速找到自己想要的优质的一手的资源。避免成为一些资源搬运者的韭菜。

书籍，音乐，电影，各种资源如何寻找，都会在本文介绍，但本文并不涉及具体哪些资源的获取。重要的是，抱着授人以鱼不如授人以渔的初衷，宝典告诉你获取资源的思路和地方，剩下的只要会敲键盘打字就可以获取到。
